Come ottenere informazioni dal documento

Da un elemento qualsiasi ad es. un paragrafo

Messaggio

    
        let h = document.getElementById('msg').innerHTML
        let t = document.getElementById('msg').innerText

Da un elemento di input testuale

    
        let h = document.getElementById('text').innerHTML // no ''
        let t = document.getElementById('text').innerText // no ''
        let v = document.getElementById('text').value // sì ''

Da un elemento di input numerico

    
        let v = document.getElementById('numero').value // no: stringa '3.14'
        let n = document.getElementById('numero').valueAsNumber // sì: numero 3.14
        // eventuale controllo
        if(isNaN(n)) alert('Not a Number!!')

Da un elemento di input tipo checkbox

            let h = document.getElementById('chk').value // no
            let t = document.getElementById('chk').checked // sì

Da un elemento di input tipo radio

            let h = document.getElementById('chk').value // no
            let t = document.getElementById('chk').checked // sì
