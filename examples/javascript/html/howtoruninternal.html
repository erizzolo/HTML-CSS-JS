Codice Javascript inserito in elementi script interni

Codice Javascript può essere inserito in documenti web tramite un elemento script, la cui posizione, non vincolata, determina l'esecuzione immediata del codice.

Questo documento contiene il seguente script all'inizio del body: <script> alert('Codice eseguito dal primo script (documento non caricato!)') </script> e quando il browser incontra l'elemento script (l'ha già incontrato!) lo esegue!!!

Questo documento contiene il seguente script alla fine del body: <script> alert('Codice eseguito dal secondo script (documento caricato!)') </script> e quando il browser incontra l'elemento script (lo incontrerà non appena chiuderete la finestra del primo alert) lo esegue!!!

Naturalmente anche questo metodo è sconsigliato in generale (come gli stili interni) salvo casi particolari
