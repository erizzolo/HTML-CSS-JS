Da un elemento di input numerico

Dopo aver ottenuto un riferimento all'elemento, si usa la proprietà valueAsNumber.

Attenzione: il risultato può essere NaN!!!

Vedi il codice

    
        let h = document.getElementById('number').innerHTML // no ''
        let t = document.getElementById('number').innerText // no ''
        let v = document.getElementById('number').value // no ''
        let x = document.getElementById('number').valueAsNumber // sì ''
    
    
    
        let h = document.getElementById('range').innerHTML // no ''
        let t = document.getElementById('range').innerText // no ''
        let v = document.getElementById('range').value // no ''
        let x = document.getElementById('range').valueAsNumber // sì ''
