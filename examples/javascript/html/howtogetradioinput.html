Da un elemento di input radio

Ciascun elemento ha la proprietà checked, ma è scomodo testarli tutti quando sappiamo che uno solo può essere selezionato.

Quindi si usa il metodo querySelector con un selettore particolare: input[name="nome_gruppo"]:checked dove:

  • nome_gruppo è il valore dell'attributo name usato per raggruppare gli input

Il significato del selettore è: gli elementi di tipo input con attributo name uguale a "nome_gruppo" e selezionati.

Dopodichè si può usare, ad esempio, la proprietà value dell'elemento.

Attenzione perché se nessuna opzione è selezionata il metodo restituisce null!!!

Vedi il codice

Scegli il tuo linguaggio preferito:

    
        let e = document.querySelector('input[name="linguaggio"]:checked')  // 1 ''
        if(e == null) { // nessuna opzione selezionata
            document.getElementById('radioNone').innerText = 'nessuna'  //  ''
        } else {
            let scelta = e.value
            document.getElementById('radioValue').innerText = scelta // ''
        }
