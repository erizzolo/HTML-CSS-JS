Codice Javascript inserito in elementi script esterni

Codice Javascript può essere inserito in documenti web tramite un elemento script, la cui posizione, non vincolata, determina l'esecuzione immediata del codice: se è presente l'attributo src esso determina il file esterno contenente il codice che sarà eseguito.

Questo documento contiene il seguente script all'inizio del body: <script src="../js/externalfirst.js"></script> e quando il browser incontra l'elemento script (l'ha già incontrato!) lo esegue!!!

Questo documento contiene il seguente script alla fine del body: <script src="../js/externalfirst.js"></script> e quando il browser incontra l'elemento script (lo incontrerà non appena chiuderete la finestra del primo alert) lo esegue!!!

Naturalmente questo metodo è consigliato in generale poichè consente il riutilizzo del codice in più pagine.
